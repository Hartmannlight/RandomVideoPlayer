# RandomVideoPlayer

RandomVideoPlayer is a small Windows app that plays all videos from a selected folder in random order.
It provides a simple GUI, remembers your settings, and supports optional logging.

## Requirements

* **Windows**
* **mpv** must be installed:
  [https://mpv.io/installation/](https://mpv.io/installation/)

## Usage

1. Download the latest `RandomVideoPlayer.exe` from Releases.
2. Run it (no installation required).
3. Choose your video folder.
4. Optional: enable program logging or playback history and select file paths.
5. Press **Start** to begin random playback.

## Features

* Randomized video playback
* Simple GUI
* Persistent settings
* Optional logs (program + playback history)
* Standalone `.exe`, no Python needed
